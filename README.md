# hw2gradingCode
1. Download this code
2. Download the submissions and unzip.
3. Move *just* the sorting .py file into the submissions folder.
4. After everything is sorted go ahead and put everything else (.py and .json files) into the submissions folder.
5. Start by running ds1grader.py
6. Open hw2_ds1_grades.csv
7. Optional
8. Add the following conditional formatting rules in:
A) Any cell between 1 .. 7 color it yellow -- they got partial credit.
B) Any cell that is 0 -- they got a 0 so you may want to check it.
